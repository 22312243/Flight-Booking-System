# Flight Booking Database System

A relational database system for managing flight booking operations, built with strutured SQL-based data management using PostgreSQL and hosted on Supabase.

### Technologies used
- PostgreSQL 
- SQL
- Supabase(Cloud PostgreSQL environment)

### Features
- Designed schema with tables for flights, passengers, bookings, airports, airlines, and payments 
- Defined primary/foreign key constraints 
- Full CRUD operations: insert, retrieve, update, and delete records

### My Contributions
- Designed parts of the relational schema
- Wrote and tested SQL queries
- Validated data operations in a shared Supabase environment
